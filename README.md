# Art Gallery Explorer: Back End

This repository contains the back end for the Art Gallery Explorer application.

Built with **Node.js**, **Express.js**, **MongoDB**, and **JWT authentication**, it provides a REST API for user authentication, profiles, and saved artwork.

## Running the Project

`npm run start` — Launch the server

`npm run dev` — Launch the server with hot reload
